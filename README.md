# C# Login & Registration System (Visual Programming)

## Description
A **desktop application** built using **C# and Visual Programming (WinForms/WPF)**.  
This system provides a **user registration and login functionality** with database integration.  
Users can **sign up, log in, and log out** securely.  

It is designed for learning **GUI programming, event-driven development, and database connectivity**.

## Tech / Tools Used
- Visual Studio (WinForms / WPF)  
- C# (programming language)  
- Laragon / MySQL (for database connection)  
- GUI Components: Buttons, TextBoxes, Panels, Labels

## Features
- **User Registration:** Allows new users to create an account  
- **Login:** Authenticate users using database credentials  
- **Logout:** Safely ends user sessions  
- **Database Connection:** Stores user information in MySQL via Laragon  
- **Input Validation:** Validates user entries (e.g., email format, password length)
